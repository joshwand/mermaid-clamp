## Key Structural Decisions

- `src/editor.ts` is a **separate entry point** from `src/index.ts`. Consumers who only want static rendering import the layout engine without the editor overlay code.
- Test files are **colocated** with source: `foo.ts` → `foo.test.ts`
- `docs/` is user-facing documentation, distinct from `_memory/` which is agent-facing
- `demos/` holds Showboat-generated demo documents for human verification
